Machine Types

Machines are divided into two types: Generators (coloured blue in Psycle's default skin) and Effects (green). Generators produce sounds, and Effects modify those sounds. To create audible sound, your generator must be connected to the Master Machine, either directly or through some Effects.

Mute: Stops all sound output from the machine.
Solo: Allows sound output only from the particular Generator.
Generator no.: The number of the generator that is used for tweaking, notes etc. Generator numbers start at 00 and can go up to 3F (in other words you can have a maximum of 64 generators).
Machine name: The name of the machine - this by default is the name of the plugin e.g. Auguru Synth 2f, but it can be changed by right-clicking the machine and entering a new name.
Panning: Modifies the left or right bias of the machine.
VU Meter: Shows the volume of the machine during playback (and rendering).
Bypass: Bypasses (skips) the particular machine during playback (and rendering).
Effect no.: The number of the effect - used for tweaking. Effect numbers start at 40 and can go up to 7F (maximum of 64 Effects).

If Psycle cannot find the correct plugin when loading a song, a dummy plugin is inserted in its place, which is denoted by an X in front of the original plugin's name.
