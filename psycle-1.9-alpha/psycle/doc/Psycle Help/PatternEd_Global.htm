Global Commands

Psycle has a set of Global commands, which fill up the parameter section of a note position:

... .. .. FXXX

FFxx Change BPM
FExx Extended Command:

FE00-FE1FSet Tick Per Beat
FE2x Set Bypass mode on/off for the specified machine (e.g. --- .. 02 FE20). Make x non-zero to bypass the machine, and make it zero to turn the machine back on.
FE3x Set Mute mode on/off for the specificed machine (e.g. --- .. 02 FE30). Make x non-zero to mute the machine, and make it zero to turn the machine back on.
FEBx Loops the current pattern x times. FEB0 sets the loop start point.
FEDx Causes a "pause" of x rows (the current row becomes x rows longer)
FEFx Causes a "pause" of x ticks (the current row becomes x ticks longer)
FDxx Delay Trigger - xx/256 ticks, which works both with note-ons and noteoffs.
FCxx Change Global/Wire Volume - the second column shows the wire number, the third shows the machine number, and the fourth shows the volume (between 0 and FF) e.g.

--- .. .. FC80    Sets Global Volume to halfway position.
--- 01 05 FC40    Sets wire 1 of machine 5 to volume 40 (in hex).

You can find the wire number in the title bar of the wire dialogue
 
FBxx Retrigger - xx/lines duration per trigger.
FAxy Retrigger Continue.
x: x/16 lines duration per trigger (0 to use current value)
y: acceleration. Values range from 1-F

         9-A-B-C-D-E-F | 1-2-3-4-5-6-7-8
slowest<- deceleration | acceleration -> fastest

This does not trigger on new line – it continues from the previous retrigger position
 
F8xx Sets the indicated machine's panning e.g.

--- .. 05 F880    Sets the panning of Generator 5 to 80 (Middle)
 

F3xx Jump to pattern position xx.
F2xx Break to line - jumps to next pattern, to the line specified by xx.
