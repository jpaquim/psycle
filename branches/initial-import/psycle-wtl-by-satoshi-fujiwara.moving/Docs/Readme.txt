Psycle Modular Music Creation Studio On WTL
------------------------------- 1.741 WTL Porting Version ------

contents:
このプログラムはPsycleをWTLにポーティングしたものです。
目的は、

・WTLはMFCの代わりになるのか移植してみることで検証する
・Psycleソースを理解する

ことにあります。
ですので、安定したものを使用したい場合は「本家」をご使用ください。
また、PsycleWTLはXPでしか動作いたしませんのであしからず。

これをベースに少しずつ改造していきながら、オリジナル部分を出していければなと思っています。
