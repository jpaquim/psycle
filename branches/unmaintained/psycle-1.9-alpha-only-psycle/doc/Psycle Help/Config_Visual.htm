Visual

(Configuration > Settings... >>Visual)

Pattern View

Draw Empty Data: Whether placeholders are shown where there are no numbers e.g. C-5 03 06 ....
Font Face:Click on the button below the text to select the Font used in the Pattern Editor.
Point, X, Y:These are the size of text, and width and height of the text spaces used in the Pattern Editor.
Line Numbers:Whether line numbers are displayed at the left-hand side of the Pattern Editor.
Cursor:Whether the cursor position is highlighted on the line numbers.
In Hex:Whether the line numbers are displayed in hexadecimal notation (like the rest of Psycle).
Skin:Allows you to select the skin to be used for the "header" above each track in the Pattern Editor.
Colours:Select the colours for different parts of the Pattern Editor, which can be a gradient. The left colour is the leftmost colour, which is interpolated left-to-right across the screen to give the colour on the right. The parts correspond as shown below.

Part: Colour For:
FontNormal Font colour
Font PlayThe note highlighted when playing.
Font SelThe font of a block selection.
Font CurThe font of the character under the cursor.
SeparatorThe gap between Tracks.
BackGndBackground Fill.
RowLines separating rows.
BeatEvery x beats (x = lines per beat).
4 BeatsEvery 4 beats.
PlaybarHighlighted bar that moves when playing.
SelectionBlock Selection fill
CursorCursor fill

Machine View

Draw Machine Indexes:Show the machine numbers in the Machine View.
Generator/Effect Font and Colour:The Generator/Effect font (blue and green boxes respectively) can be set by clicking the button (set to "Tahoma" in the diagram), set the font size in the drop-dwon box and click on the "Generators"/"Effects" button to set the text colour.
Background:Select the background colour.
Wire Width:Set the width of the wires that connect machines.
Wires:Wire colour.
Antialias Halo:Set the thickness of the colour smoothing that surrounds the wires.
Polygons:Colour of the triangles that are in the middle of the wires.
Polygon size:Size of these triangles.
Skin:Select the machine's skin.
Background Bitmap:Set the background image for the Machine View

VU Colors

Set the Background, VU bar and the OnClip (when the level gets about 0 dB) colours. The checkbox determines whether the VU meters are drawn - if you want to save CPU usage, untick this box.

Global Options

Enable Graphics double buffering to have a cleaner view. This will make more difference with a slower CPU, however it will also use CPU power.

Theme Options

Load and save themes using the buttons.

Native Gui

You can change the following settings for the text color, and for the background color:
Title:Colours used in the Group title.
Param:Colours used in the Parameter name line
Value:Colours used in the Value line
SelParam:Colours used for the Parameter line, when selected
SelValue:Colours used for the Parameter value, when selected.
Dial Bitmap:Bitmap for the dial - see Skinning
