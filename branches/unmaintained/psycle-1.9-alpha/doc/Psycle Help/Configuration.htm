Other Settings

AutoStop Machines (Configuration > AutoStop Machines) – All machines not producing any output will be automatically turned off (and on again). This can significantly reduce real-time CPU usage. Currently this causes problems with certain delay-type machines: the machine is turned off before it has completed the delay, so the delay does not continue as long as it should. This error will not appear in the rendered version of your song.

Loop Playback (Configuration > Loop Playback) – Song playback will be looped continuously.
