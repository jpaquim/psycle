Gear Rack

The Gear Rack is a convenient way to organise your machines.

Generators Tab/Effects Tab

The Generators or Effects are displayed in numerical order. The buttons function as follows:
Create/Replace: Creates (or Replaces, if already exists) a machine in the currently selected position.
Delete: Deletes the selected machine.
Parameters: Displays the machine's Parameters window.
Properties: Displays the properties window.
Exchange: Swaps the positions, connections and machine numbers of two machines that have been selected in the list.
Clone: Makes a copy of the selected machine with all its settings. If you select a second position in the list, the copy will be placed there, otherwise it is placed in the first empty position.

Instruments Tab

Same functions as above (applied to Instruments used in the Internal Sampler) except that instead of the Properties button there is the Wave Editor button.
