MIDI Controllers

(Configuration > Settings... > MIDI Controllers)

This screen allows to change the meaning of the MIDI messages received by Psycle and do a different action instead.

The two dropdown boxes of Set Generator and Set Instrument configure in which way Psycle will know the generator and/or instrument that is going to receive the message.
Example: If it is set "by Program change", then the messages received in the same MIDI channel will be sent to the machine indicated by the MIDI Program Change message received. (This is the original behaviour of the Synchronized mode)
If it is set at "Currenty selected in Psycle", then, any message received will always go to the generator and/or instrument selected in the machine dropdown box in the main toolbar. (This is the original behaviour of the Immediate mode)

The setting for Record Raw MIDI as mcm does not try to apply any of the controller mappings below (except the Velocity map) and sends instead an mcm command directly to the machine, who is responsible to understand it.

The Map velocity option tells which command to use for the note on command as well as the velocity MIDI command.

The Map pitch option allows to map the Pitch message (the wheel in many MIDI keyboards) to a command.

The rest of the Maps are user configurable and will have an effect only if the "Record Raw MIDI as mcm" is unchecked.

To map a command to a parameter, type its number in the leftmost box (this is the MIDI CC number - read your synthetiser's manual for more details). Then, select the function you want to use, and specify the minimum and maximum values of the function (this means that your controller's input will be scaled between the two values). They behave as follows:

Function Translates the MIDI CC into:
cmd Internal machine commands - read your plugin's manual for details.
twk/tws Tweak/Tweak Smooth Smooth - the value to the right of the drop-down box is the parameter number.
mcm Raw mcm command.

Note: The Synchronized mode does also offer a dynamic mapping set by sending MIDI commands. See the CC 126 and CC127 on the MIDI chart
