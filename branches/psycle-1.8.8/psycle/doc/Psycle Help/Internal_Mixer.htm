Mixer Machine

The send-return mixer is a very adaptable machine that allows to work with Psycle in a completely different way than what you're used up to now.
It adapts to the number of wires connected, so it doesn't use any more screen space than the needed

In its most simple way you can use it to control the volume of up to 12 signals (like if it was another master machine). But even then, it is more functional, giving panning per channel, vu-meters and separated gain from volume slider.

In its intermediate functionality, it lets you have up to 12 effects connected to it, and send part (or all) of each of the input channels to them, and mix them all at output, controling the volume of the audio returning from the effects.

Finally, in its most complex way of working, you can use several effects in chain in one send, route the output of one send to another, and almost convert any song done with connecting machines between them, to use the mixer.

Its layout is very simple. If you don't have effects, there is only one column for Master volume (leftmost), and each of the input channels.
Each channel has a knob for "Mix" (What is sometimes called "dry signal"), for Gain (this value is mapped to the volume of the wire), and for panning, followed by the volume slider and four small buttons as follow:

  • S: Solo this channel. Only mix the audio of this *input* channel. It does not mute the returns of the effects.
  • M: Mute this channel. Don't mix the input of this channel. It also prevents its audio to be sent to the effects.
  • D: Mix Dry component only. Doesn't send the audio of this channel to the effects. (Sets the "send" knobs of this channel to 0% internally)
  • W: Mix Wet component only. The audio is sent to the effects as normal, but the dry component (Mix Knob value) is muted.
Master Column has a knob named "D/W" in the place of the Mix knob. This allows you to control how much of the dry (coming from input channels) and wet (comming from the effects) is mixed to the output.

When adding effects, they are added as a column at the right side of the window, keeping input channels on the left. For each one you add, a new line is added to all input channels that allows to specify which percentage of the signal is sent to the send effect.
Each one has a slider and some buttons in the place of the knobs that allow to route the audio output like this:

  • Master: There's one button that says Master. If it is lit, the output of this effect is mixed directly to the output of the Mixer machine.
  • Route : As more effects are being connected, buttons with the word Route appear. If one of them is lit, the output of this effect is used as the input of the send that corresponds to the line of the button.

The buttons next to the slider do the following:
  • S: Solo this return. Allows to listen to this return alone. It does mute all the other returns *and* the dry signals of all the input channels.
  • M: Mutes this return. Mutes the audio that returns from this effect, not mixing it. Note: If you mute the only effect that outputs to master, you won't hear anything. You can't use this as a "bypass".

Almost every part of the interface can be tweaked from the patterns. The following table explains which values you can use for this task:

[0x]:
	0 -> Master volume
	1..C -> Input volumes
	D -> master drywetmix.
	E -> master gain.
	F -> master pan.
[1x..Cx]:
	0 -> input wet mix.
	1..C -> input send amout to the send x.
	D -> input drywetmix. ( 0 normal, 1 dryonly, 2 wetonly  3 mute)
	E -> input gain.
	F -> input panning.
[Dx]:
	0 -> Solo channel.
	1..C -> return grid. // the return grid array grid represents: bit0 -> mute, bit 1..12 routing to send. bit 13 -> route to master
[Ex]: 
	1..C -> return volumes
[Fx]:
	1..C -> return panning

Some things to know:

  • If you delete a wire to a mixer, the number of columns will only decrease if it was the last input, or last effect.
    In plain words: If you have three inputs and delete the wire that corresponds to the first input, there will still be three columns, but the first one will be inactive.
    This behaviour is done to avoid any tweaks done in the pattern to become automatically erroneous when changing the machines connected.
  • You can move the columns around. Drag the title of any column left or right to put it where you need it. Note that you are not allowed to interlace inputs and effects.
