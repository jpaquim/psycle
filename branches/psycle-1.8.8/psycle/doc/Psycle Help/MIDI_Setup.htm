Basic Setup

Psycle has support for MIDI input events, which provides two features: enter data into the patterns and control Psycle in realtime. MIDI messages can come from other software using a virtual MIDI device or from a MIDI keyboard or controller.

MIDI note on/off messages can be mapped to any generator and instrument across all 16 MIDI channels. Selecting the instrument allows to use it with the Sampler or other machines that use the auxiliary column. The parameters of Psycle Plug-ins and VST instruments and effects can be controlled by using MIDI continous controller (CC) messages.

The basic setup of MIDI in Psycle consist of using the Immediate mode and configure the selection of the generator and instrument as the one selected in Psycle (See the controller's tab in the configuration dialog).
A slightly more complex but also more flexible is using also the Immediate mode, but configure the Generator to be changed by MIDI Program change, and Instrument by channel index. This scenario is perfect to play with the Fluidsynth native plugin and other VSTis
