The Sequence Editor

The Sequence Editor is where you organise your patterns, and is located on the left side of the Machines window (F2). The song progresses from top to bottom, and each line in the list holds one pattern. The number on the left is the position number, and the number on the right is the pattern number. You may use the same pattern in more than one position.

Plus/Minus – Increase/Decrease the pattern number at the selected position by one.

Double Plus/Minus – Increase/Decrease the pattern number at the selected position by 16 (10 in hexadecimal).

New – Create a new pattern with the default pattern length in the next position after the current selection.

Clone – Copies the current selected patterns and pastes them with new pattern numbers.

Ins – Inserts the current selected pattern below the current selection, moving the rest of the patterns down by one.

Del (Edit > Sequence Delete) – Deletes the pattern from the list, moving the other patterns up by one. Note: This does not delete the contents of the pattern.

Cut (Edit > Sequence Cut) – Cuts the selection to the internal clipboard.

Copy (Edit > Sequence Copy) – Copies the selection to the internal clipboard.

Paste (Edit > Sequence Paste) – Pastes the selection after the currently selected position.

Clear – Clears ALL sequence and pattern data - this WILL erase all your patterns.

Sort – Sorts all visible patterns by changing their numbers to match their position. If a pattern appears multiple times, its number will be assigned based on the first instance of it e.g. if pattern 03 is in positions 04, 07, and 21, it will be renamed to 04.

Length – Alters the length of the song in patterns. Reducing the size of the song with this method can only be reversed using the Undo comand. Increase the size of a song with this method will increase the number of positions, which will be filled with pattern 00.
