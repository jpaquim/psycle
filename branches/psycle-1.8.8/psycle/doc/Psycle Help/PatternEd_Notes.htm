Entering Notes

Notes can be entered via the computer keyboard or a MIDI keyboard . The actual octave that is entered into the pattern editor is set in the Song Bar , and can be increased or decreased by pressing "*" or "/" (numeric keypad) respectively.

Psycle uses the English notation for the note names (See Wikipedia to see the correspondence with other notations). This notation uses the first 7 letters of the alphabet as notes.

Notes in psycle look like this:

C-4 01 00 0C3F
 1     2    3     4
1. Note name, which can range from C-0 to B-9. The number represents the octave - C-4 is usually the same as middle C on the piano (this can sometimes depend on the sampler/synthesiser settings).
2. Auxiliary column - This column has two uses, depending on the type of note. If it is a tweak, tweak smooth or MIDI command, it determines the parameter or command to be sent. When it is a note, it depends on the machine. For the Sampler Machine, it indicates which instrument is going to play.
With VST's, it can be used (it is not required) to specify the MIDI channel number (from 0 to F). Other machines like fluidsynth or Yannis MIDI can use this column too to indicate in the same way.
3. Machine number (in hexadecimal).
4. Note parameters - these will depend on your machine. For VSTs, the 0Cxx command sets the volume at note on, where xx is the volume (from 00 to FF). For native machines and the internal sampler, refer to their documentation.

When entering notes, make sure the cursor is in the left-hand side of the track. To edit other parts of the note entry (channel, machine number, parameter) move using the arrow keys or the mouse.

To enter chords, hold shift and press each note. They will be placed on the same line, to the right of the cursor.

Unlike other notation systems, note durations will depend on the volume envelope settings in the machine, and on note-off commands. Note-off commands send a "note off" signal to the machine, which often triggers the "release" part of the volume envelope (this depends on the machine itself). You can enter them using the key "1" over the "Q" key ( on a QWERTY keyboard). It should look like this:

off .. XX ....

where XX is the machine number.

Omitting the machine number will send a noteoff command to the machine that was previously playing in this track. Hitting Ctrl+1 will enter one of such off commands.
