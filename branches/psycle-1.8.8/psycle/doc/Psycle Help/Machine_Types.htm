Machine Types

Machines are divided into two types: Generators (coloured black in Psycle's default skin) and Effects (blue). Generators produce sounds, and Effects modify those sounds. To create audible sound, your generator must be connected to the Master Machine , either directly or through some Effects.

Mute: Stops all sound output from the machine.
Solo: Allows sound output only from the particular Generator.
Machine number: The number of the generator or effect that is used inside the pattern editor. Generator numbers start at 00 and can go up to 3F and Effect numbers start at 40 and can go up to 7F (they are expressed in Hexadecimal numbers, so there are up to 64 machines of each type).
Machine name: The name of the machine - this by default is the name of the plugin e.g. TB 303, but it can be changed by right-clicking the machine and entering a new name.
Panning: Modifies the left or right bias of the machine.
VU Meter: Shows the volume of the machine during playback. It is the RMS value expressed in dB, where the bottom is -40dB and the top 0dB
Bypass: Bypasses (skips) the particular machine during playback (and rendering).

If Psycle cannot find the correct plugin when loading a song, a dummy plugin is inserted in its place, which is denoted by an X in front of the original plugin's name.
