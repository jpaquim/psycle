Keyboard and Misc. Tab

(Configuration > Settings... >Keyboard and Misc.)

To change the keymap settings, select the action in the listbox, click inside the "Key:" textbox, then press the new key combination.

Behaviour

Right CTRL = play.
Edit Toggle = stop:		 Causes the right Control key to be used to begin playback from the current sequencer position, and the Record button/key to stop playback.
FT2 Style Home/End Behaviour: When unchecked, pressing home/end will move the cursor to start/end of row (once), start/end of track (twice) and start/end of pattern (3 times),When checked, pressing home/end will move the cursor to the beginning/end of the track.
FT2 Style Delete Behaviour: This applies to the "Delete Row" command (usually backspace). FT2 style deletes the note above cursor, while normal style deletes the note that the cursor is on.
Windows Behaviour on selected blocks This function enables the selections to act like they would do in a Word processor in windows. You can select them, drag them to move them, and when clicking they get unselected. Else, they will behave like in classic trackers.
Use Shift+Arrows for Selection: Allows you to use shift and arrows to make selections in the pattern editor (much like in a word processor).
Wrap around at first/last line/track: When checked, this allows the cursor to go back to the top/left of the pattern after reaching the bottom/right and viceversa (when navigating with arrow keys).
Cursor always moves down in effect column: Cursor always moves downward (instead of sideways as it normally does in some columns).
Record Mouse Tweaks as tws (Smooth Tweaks): Records tweaks made in real-time as smooth tweaks.
FT2 Style Record Behaviour in Unarmed Track: If there are no tracks with recording enabled, and you play notes/enter commands during playback, with this option data will be entered where the cursor is.
Force pattern step 1 when moving with cursors: Ignores pattern step setting and moves down/up/left/right one row/track in the pattern editor when navigating using the arrow keys.
Page up/page down step by Lets you configure the jump that page up and page down keys will do in pattern lines.

Miscellaneous Options

Autosave current song after (mins): Automatically saves the song to "autosave.psy" (in your Psycle Songs directory) after the specified number of minutes.
"Save File?" feminders on load, new or exit: Asks the user if the current unsaved file is to be saved upon loading or exiting.
Show song info on load: Displays the Song Properties information when the song is loaded.
Allow multiple instances This option lets you specify if you would like multiple psycle windows, or would want just one.
Store settings Indicates where the settings are to bestored. Use this to save the settings to other places than the registry. When loading Psycle, it will first look for config on .exe directory, else it will try in the locap AppData, and at last will check in the registry.
Default New Pattern Lines: Sets the default number of lines per pattern (between 1 and 256).
