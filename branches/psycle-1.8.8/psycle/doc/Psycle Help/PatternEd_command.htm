Mcm Command

The Mcm command has evolved to have two different functionalities:

As MIDI command

The Mcm command works in a similar way than the Twk Command:

If the following is a pattern Row, 
Description: (1) (2)(3) (4)
Pattern Row : ---- .. .. ....
then, on (1) , you insert the Mcm command.
To do so, you should press Mcm key. Check and/or change its value in the keyboard configuration dialog).

On (2) , you define the Midi command number that you want to send.
You might need to check some MIDI documents and/or the Plugin's help to know what to write in here. These values are always from 80 to FF.

On (3) , you specify the number of the Machine, just like you do with tweaks.
This number is the one shown in the combobox, or in the machine's dialog.

Finally, on (4), you specify the Value to send for this midi command.

(4) is divided in two groups of two values AABB. You might only need to speficy AA.

Usually, you'll use it to send MIDI CC's. CC Command is "B0".
Example, this sets the machine volume to half of it: 
Mcm B0 01 0740
B0 Send a MIDI CC (Control Change) command to the MIDI channel 0
01 of the Generator number 1
07 In order to change it's (Channel) volume
40 to 40 (in a scale of 0 to 7F, HEX)

As command redirector

The Mcm command can also be used for another purpose. It allows to send a command to another track. This way, you can workaround the limitation where you can only put one of them per track.

Syntax is as follows:

If the following is a pattern Row, 
Description: (1) (2)(3) (4)

Pattern Row : ---- .. .. ....

on (1) , you insert the Mcm command.
on (2), insert the (pattern) track number where the command will be sent to (This number will be at much 3F).
on (3), insert the machine number
on (4), insert the command that you want to send.

Example: Send command 0C50 to track 02 of machine 06: 

 mcm 02 06 0C50
