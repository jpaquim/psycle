Input/Output

(Configuration > Settings... > Input/Output)

Audio I/O

Depending on your system and soundcard, you will get several different options for sound output. Options include Silent (no output), Windows waveout MME, Directsound, ASIO (if supported) and WASAPI (If Vista or 7)

By clicking the "Configure..." button, you can then set device/soundcard to use, the sample rate and bit depth (depending on setting). You can also set the buffer length, in total samples or in blocks of buffers depending on driver. These will determine the latency (the delay in playback). If you find your audio skips and breaks unnecessarily, you may need to increase these values.

Note: Most ASIO drivers, as well as WASAPI exclusive, will only allow sound to be played by Psycle, dissallowing other applications to use the same soundcard at the same time.

MIDI I/O

Select the MIDI input device from the dropdown list. This will be the main device. Sometimes there will be more than one port for your hardware, such as a MIDI through port and/or a data port.

If you are only using a MIDI keyboard or controller to send notes and tweaks to Psycle while composing, you might find easier to use the Immediate Playback option. This doesn't need a MIDI clock and there is no synchronization to the audio.

On the other hand, if you want to do so in realtime, using Psycle as a softsynth, it is recommended that you setup a clock (which can be sent by the same MIDI device or another one), and messages will get queued and played in synchronization with the audio.
This second method is covered in the MIDI Tutorial

Both methods support receiving messages from multiple MIDI channels, route them to different machines and remap the messages. The main difference is how they are processed by Psycle.
