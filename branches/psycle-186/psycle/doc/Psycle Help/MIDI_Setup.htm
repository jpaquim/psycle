Basic Setup

To use Psycle with a MIDI sequencer, you will need to set up a Virtual MIDI port (a software emulated MIDI input/output port). You will need software such as Hubi's Loopback or MIDI Yoke
If you're using Windows 2000/NT/XP you will need MIDI Yoke, since Hubi's Loopback only works on previous versions of windows.

After installing your Virtual MIDI Port, select it as the Output from your MIDI sequencer, and as the Input in Psycle.

In the Psycle Configuration Window, select the input device from the list. Make sure that it will match up with the output device selected in your MIDI sequencer i.e. with MIDI Yoke, if you select MIDI Yoke 1 as Output in the MIDI sequencer, you should select MIDI Yoke 1 as Input in Psycle.

If you are using a separate device for MIDI Sync (see later for details) then you can select this from the Sync Device listbox, although if you plan to send a MIDI Sync clock through the same device as the main input device then you can leave this unselected (None). This allows the MIDI to flow as follows:

Sequencer MIDI Out > Virtual MIDI Port > Psycle MIDI In
