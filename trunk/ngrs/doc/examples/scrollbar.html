NScrollBar

Short Description:

The NScrollBar component is used for vertical and horizontal scrollbars. Automatically the component is able to scroll another Visualcomponent.

Methods

setRange/range --doubleXdouble : the minimum, maximum position value
setPos/pos --double : the position of the Thumb relative to the range
setSmallChange/smallChange--double : position change when pressing the scrollbar buttons
setLargeChange/largeChange--double : position change when pressing the area above/below the scrollbar thumb
setOrientation --nVertical/nHorizontal : the orientation of the scrollbar
setControl --NVisualComponent*X(nDx/nDy): automatic scrolling of children components and graphic gc of controlComponent. nDx scroll is horizontal, nDy scroll is vertical. range is set to the clientWidth of the component.

Events

change: is emitted at setPos, down/up button press, above below thumb press, thumb scroll scroll: is emitted at down/up button press, above below thumb press, thumb scroll
