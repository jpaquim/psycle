MIDI Controllers

(Configuration > Settings... >>MIDI Controllers)

This is where you can map external (or virtual) MIDI controllers to control certain parameters. Make sure that your device is selected as the MIDI input device.

This window allows you to map the MIDI velocity (volume), the pitch wheel and MIDI Control Codes to various parameters. If an external sequencer is being used, the MIDI channel of the signal will determine the machine it plays on (e.g. if information is sent on MIDI Channel 15, it will act on machine number 0E). If a MIDI keyboard (or similar) is being used, the MIDI commands will act on the currently selected machine.

To map a command to a parameter, type its number in the leftmost box (this is the MIDI CC number - read your synthesiser's manual for more details). Then, select the function you want to use, and specify the minimum and maximum values of the function (this means that your controller's input will be scaled between the two values). They behave as follows:

FunctionTranslates the MIDI CC into:
cmdInternal machine commands - read your plugin's manual for details.
twk/twsTweak/Tweak Smooth Smooth - the value to the right of the drop-down box is the parameter number.
insmcm command that changes the sample selected (if the machine is the Internal Sampler).
mcmRaw mcm command.

Note: By selecting "Record Raw MIDI as MCM", the settings above are overridden and all MIDI input is recorded using the mcm commmand.
