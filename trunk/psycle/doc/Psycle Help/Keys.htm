Default Keyboard Settings

Almost all of these keys can be changed from the Keyboard and Misc. configuration tab. To reset the keyboard settings, click the "Default" button in this window, or delete this file:

<Your Windows Directory>\PsycleKeys.ini

Note Layout on the PC Keyboard

This image shows the default layout of the notes. Hover the mouse over the image below to see how the notes correspond to letters on the keyboard (English QWERTY layout). The octave coloured in blue is the one set as the current octave.
You may need to modify this for other keyboard layouts ( French AZERTY, for example ), or for your own needs (having 3 octaves instead of just 2 and a half)
