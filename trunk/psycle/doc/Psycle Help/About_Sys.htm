System Requirements

Due to the characteristics of Psycle, the processor and memory usage are highly dependant on the complexity of the song. Here you have some guidelines on what you can expect. RAM usage is expressed as memory used by Psycle alone

  CPU Speed RAM Operating System
Minimum Processor with SSE2 16MB Windows XP
Usual Processor with 64bit support (x86-64) 64MB Windows 7

Currently, there are two versions. a x86 version and a x64 version. There are two installers, and both can be installed (x64 only if you have a x64 OS)

  • x86, SSE2 support required:
    AMD: Athlon 64, Athlon FX.
    Intel: Pentium 4, Celeron Pentium 4, Pentium M, Centrino and newer
  • x64, 64bits processor needed
    AMD: Athlon 64 or newer.
    Intel: Pentium Core, Core 2 and above. (x86-64. This excludes IA-64).

No special soundcard is required to run Psycle. If you intend to play softsynths in realtime, a soundcard with ASIO drivers is recommended as these generally have low latency. There is support for WASAPI exclusive mode in Vista and 7 for low latency also.

A monitor with 1024 * 768 resolution, and at least 16-bit depth is recommended. Smaller monitors may not show all of the user interface (you miss out on a few of the checkboxes down the bottom of the pattern sequencer, but the toolbar can be detached).
