System Requirements

Due to the characteristics of Psycle, the processor and memory usage are highly dependant on the complexity of the song. Here you have some guidelines on what you can expect:

CPU Speed RAM
  Windows 95/98 Windows 2000/ME Windows XP
Minimum: A processor with SSE instructions (Pentium III or Athlon XP) 64MB 96MB 128MB
Recommended: Pentium 4 or better 128MB 256MB 512MB

No special soundcard is required to run Psycle. If you intend to play softsynths in realtime, a soundcard with ASIO drivers is recommended as these generally have low latency.

A monitor with 1024 * 768 resolution, and at least 16-bit depth is recommended. Smaller monitors may not show all of the user interface (you miss out on a few of the checkboxes down the bottom of the pattern sequencer, some of which can be accessed in the Configuration window).
