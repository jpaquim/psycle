Instruments

An instrument is essentially a sound sample that can have some parameters like envelopes and loops, and is played by one of the two sampler machines: classic sampler or Sampulse. Since the features of the two samplers are different, there is an instrument editor for the sampler and a different Instrument editor for Sampulse.
There is also a basic wave editor

All of the instrument functions can be accessed from the right half of the machine bar. Once you've added a sampler or Sampulse machine using the Add Machine dialogue window, to work with instruments you should make sure that the machine is selected, and that "Instrument" is selected in the third drop-down box.

The next drop-down box dispays the instruments that have been loaded (which are then stored within your psycle file). They are numbered (in Hexadecimal) from 00 to FE (i.e. you can have a maximum of 255 different samples in your song).
It should be noted that the instrument list shown when selecting a Sampulse machine is different than when selecting a sampler machine.

Since Psycle 1.11, this droplist doesn't show all the positions by default.
To add a new position, use the right arrow to increase beyond the visible positions.

Instrument Settings

Load – Loads a sample file from disk into the currently selected instrument position.

Edit – Opens the Sampler Instrument Editor window, or the Sampulse Instrument Editor window, depending on which machine you have selected.

Wave Editor – Opens the Wave Editor.

Virtual generators for instruments

Instruments need a sampler or sampulse machine to be used, but this requires to specify both in the pattern, and if there are several sampler machines, sometimes it can be tedious to select both correctly
With the idea to simplify this, there is a new feature in Psycle intended to use instruments in patterns as if they were a machine. Go into the chapter about virtual generators to learn how to use it.
