Wave Editor

To access the wave editor, make sure your instrument is selected in the machine bar, then click on the Wave Editor button, or from the Samples bank in the instrument editor

To add Loop points, hold Ctrl and left click on the waveform to set the loop starting point, and hold Ctrl and right click to set the loop end point. To add sustain loop points, hold Shift and left click on the waveform to set the loop starting point, and hold Shift and right click to set the loop end point.
To turn loop points on and off, use the Instrument Editor.

Menu Functions

Process

Amplify Selection... – Opens a dialogue box with a slider that allows you to modify the selection's volume.

Fade In/Out – Fades the selection in/out.

Insert Silence.. – Adds an user specified amount of silence at the beginning, the end or at cursor position.

Normalize – Amplifies the selection to the maximum possible without clipping.

Remove DC – Removes any offset from the centre (remove DC bias).

Reverse – Reverses the sample.

Edit

Select All – Selects the whole waveform.

Cut – Cuts the selection to the internal clipboard

Crop – Removes everything outside the selection.

Copy – Copies the selection to the

Paste – Pastes the waveform from the internal clipboard. The different methods allow to add, replace, mix or add while crossfading.

Delete – Deletes the selection.

Snap to zero – Puts the cursor only in samples that are at the value of zero-crossing (i.e. the middle).

View

Zoom In/Out – Scales the view down/up. You can also use the buttons and the slider on the bottom right side of the editor window.

Zoom Selection – Scales the view to fit the current selection.

Show All – Scales the view to show the whole sample.

Visual representation – Changes the way the waveform is drawn. They use the same algorithms than the sampler. Not reconstructed can help to see the wave points, and Reconstructed is the most accurate representation.

Convert

Convert to mono – Converts a stereo sample to mono by mixing the left and right channels and reducing the volume.

Toolbar buttons

Play from start – Plays the sample from the start, ignoring the cursor position

Play from cursor – Plays the sample starting at the cursor positiont

Release note (stop sustain loop) – When playing and if the sample has a sustain loop, pressing this button will act as a note off (release)

Stop note – When playing, this button stops it immediately

Move cursor to start/loop – This button moves the cursor to the left. If the sample has any type of loop, it will also stop on the loop points.

Move cursor to end/loop – This button moves the cursor to the right. If the sample has any type of loop, it will also stop on the loop points.

Load sample – Opens the load dialog to load a sample in this position.

Save sample – Opens the save dialog to save this sample to disk.

Droplist and arrows – Allows to select the sample that the editor shows. This droplist doesn't show all the positions, so if you want to add a new position, use the right arrow to increase the list entries.

Status bar

Volume bar – The blue bar on the bottom left allows to adjust the playback volume. It is set by default to the sample global volume.

Scroll bar – The center scroll bar allows to move the visualization.

Zoom bar – The two buttons and small bar on the right is the zomm bar, to increase or decrease the zoom ratio.

Popup menu

Zoom to selection – Zoom so that what is selected fills the whole window.

Zoom in – Increase the zoom.

Zoom out – Decrease the zoom.

Cut/Copy/Paste – Copy and paste operations.

Delete – Remove the selected portion.

Set loop to selection – Create a new loop that corresponds to the selected region.

Set sustain to selection – Create a new sustain loop that corresponds to the selected region.

Unset loop – Remove the loop.

Unset sustaion loop – Remove the sustain loop.
