NEdit

Short Description:

The NEdit component is a single line edit control. As data it holds a std::string.

Methods

setText/text --std::string : setting and getting the string
setPos/pos --unsigned int : sets the cursor pos.
selText --std::string : gets a evtl. selected text.
setReadOnly -- bool : deactivates the keyboard functions
setVAlign -- int : sets the vertical text alignment : values nAlTop, nAlBottom, nAlCenter
setHAlign -- int : sets the horizontal text alignment : values nAlLeft, nAlRight, nAlCenter

Events

keyPress -- const NKeyEvent & : is fired at keypress
alternativ override : void onKeyPress(const NKeyEvent & keyevent);
