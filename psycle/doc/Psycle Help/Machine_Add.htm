Adding Machines

(F9 or double click the background of the Machine Editor or the new machine button of the toolbar or View > Add Machine)

This opens the New Machine dialogue (Machines can also be added from the Gear Rack, the Machine properties and from the Wire dialog ).
Gear Browser: Displays all the plugins that are available. Select one and click OK to add it to the Machine Editor.
Machine properties: Displays the machine's details, and allows you choose whether to display a warning if a dangerous plugin tries to load.
List machines by: Displays the plugin machines in the Gear Browser by the Plugin Name or by the DLL filename.
Group machines by: Groups the machines depending on its role (Generators, Effects) or host type (Internal, Native, VST)
Load new blitz and gamefx: This is a song loading compatibility option. gamefx and blitz have a new dll name now and are incompatible with the old version. This option tells the loader to use the new dll instead of the old one. This should only be necessary for songs made with the 1.8.8 beta (or by having used the unofficial release of the plugin). Once saved the dll name will be either the older of the newer and the option will not matter.

Editing Machines

Double click the machine to bring up its Parameters window or press Shift+Enter. Alter the properties of the machine by right clicking it.

Removing Machines

To remove a machine, right-click the machine and click the "Delete Machine" button (Machines can also be deleted from the Gear Rack ).
