Machine Parameters

(Double click the machine or press Shift+Enter)

Internal Machines

Internal machines have a custom UI. You can see a detailed description of each one by visiting the Internal Machines page.

Native Machines

The parameter dialog for native machines consists of a window with knobs, with the name and value at its right. The Machine number and the name are displayed in the title bar.

They can be edited by dragging the knob up or down to increase or decrease the value respectively. To make value selection easier, holding the Ctrl key while dragging a knob will make a change 10x smaller, and holding the Shift key will change the knob by the smallest value.
Optionally, you can right click on the knob or on the text. This will prompt an edit box where you can insert the exact value in numbers.
Finally, if you doubleclick a knob, it will get reset to its default value.

VST(i) Machines

The parameter window of VST's is most of the time machine dependant, so if you have any doubt about its usage, check its manual. Some of them, though, will look like Native plugins. For those, tweaking works like explained above.

Menu, toolbar and parameter list

The menu functions are explained below:

Programs
Open(Save) Preset: Use this option to load(save) an .fxb or .fxp file containing preconfigured programs for the machine. .fxp will be loaded(saved) over(from) the currently selected Program.
This option is always disabled for native plugins.
Select Bank: Use this menu to select the active bank. A bank is a group of programs that the plugin offers to the user. There is also the User bank, which shows the presets saved by the user in the bank manager.
Select Program: Use this menu to select the active program. You can alternatively use the Combobox, or the arrows next to the combobox.
Randomize: Randomises the parameter values (be careful with your speakers, this can give some crazy results - or no sound at all).
Reset Program: Resets the program to the machine defaults.
This option is only available to Native machines.
Activated: You can use this option to mute this generator/effect. Alternatively, you can use the Power button in the toolbar.
Views
Parameter List: This option shows a window with a list of all the parameters of the machine of which their value can be set. The slider next to the list allows to change the value of the selected one. And you can use it also to know the parameter Index to use for tweaking from the Pattern Editor.
Bank Manager: This lets you manage the programs of this machine. Note: Only user presets can be modified from here(i.e. those stored in the path configured in the configuration)
Toolbar: This shows or hides the toolbar.
About: This button is only enabled in Native machines and is machine dependant. Some machines show parameter information, others do an action like asking for a file.
Machine info: Shows information about this plugin.

Note about the banks and presets:

Psycle is able to show banks and programs from three different places: The plugin's internal data, a preset file in the same directory and same name than the plugin, and a preset file in the presets directory of the configuration dialog.

Psycle selects by default the bank placed in the same directory than the plugin if it exists. This would seem to hide the real programs, so use the feature with care.

Note also that only the presets located in the presets directory can be modified with the Bank manager
