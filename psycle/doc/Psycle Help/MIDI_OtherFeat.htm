Other features

When the pattern view is active in Psycle then all MIDI note-on messages are entered into the pattern. It is possible to play generators with the tracker interface and control the generator parameters with external MIDI controllers (e.g. using Phatboy hardware device). Remember the controller->generator parameter must first be set (using CC 126/127).

For more on this, see the MIDI Controllers section of the Configuration Window.
