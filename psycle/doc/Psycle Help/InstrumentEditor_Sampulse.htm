Instrument Editor for Sampulse

Use the list on the top left to navigate between the different instruments, and use the tabs to change between the different parameter windows. Each one is described in detail below.


The list on the bottom left allows to see which wave samples are used by the selected instrument.

Instrument List Buttons

Load – Loads an instrument file from disk into the currently selected instrument position.

Save – Save the currently selected instrument to disk.

Duplicate – Creates a new instrument with a copy of the currently selected one

Delete – Deletes the current instrument. If it has samples, it also asks if it should delete them too.

Pitch tab

Instrument name – Changes the name of the instrument.

Virtual generator – Use the Virtual generator for instruments.

Set default mapping – Sets the default settings of note and sample for the note mapping of this instrument.

Set Sample – Sets the indicated wave sample (a popup menu appears when clicking on it) as the sample for all the notes of this instrument.

Edit mapping – Opens the Note mapping editor. See below

New Note Action – Whether to cut, release, start fading or do nothing (continue) when another note from the same machine is triggered while the previous note is still playing.
It is possible to indicate an alternate action in case of the same note, sample or instrument

keyboard – Offers a visual representation of the mapping, and allows to modify it note by note, when clicking over one.

Envelope

Sampulse offers a freeform envelope editor, able to define multiple points, as well as defining sustain and continuous loops. The editor can be edited directly dragging with the mouse. The right click (context) menu allows to add, remove and set the points and loops

Envelope enabled – Activaes or deactivates the envelope. Note: Envelope can be temporarily de/activated from the pattern with the 0EEx command (See tweakings.txt of the help in Sampulse machine)

Carry (continue) – An envelope that has carry enabled will not start from the start if a new note comes while the previous note was still active (i.e. not stopped nor doing noteoff).

Millis and Ticks – Configures the resolution of the envelope. When working in millis, the envelope lasts the same whenver the tempo changes. With ticks, the envelope duration changes when tempo changes.

ADSR and freeform – Configures the way that the envelope can be edited. ADSR mode defines an Attack-decay-sustain-release envelope, and the six sliders below the envelope can be used to modify it. In freeform mode, the sliders are disabled, and the envelope can be changed and edited with the mouse.

Sustain – Defines the sustain loop points. A sustain loop loops only until a noteoff is received. The loop begin and loop end can be the same or different points.

Loop – Defines the continuous loop points. A continuous loop loops until the note stops. The loop begin and loop end can be the same or different points.

Base – For an ADSR envelope, allows to move up or down the start and end points.

Modulation – For an ADSR envelope, allows to move up or down the attack and sustain points.

Attack – For an ADSR envelope, allows set the attack time.

Decay – For an ADSR envelope, allows set the decay time.

Sustain – For an ADSR envelope, allows set the sustain amount.

Release – For an ADSR envelope, allows set the release time.

Note mapping editor

Sampulse instruments can use more than one sample at a time, and to do so, they use the note mapping, which means that each of the possible notes, will trigger a specific note and sample number.
In the capture, the note F-3 will trigger the sample 0B and play it with the note F-3 as well.

The window is divided in three distinct zones.
On the left there is the list of maps. First is the note mapped, next the note that will play instead, and the third is the sample number.
On the top right there is the list of the possible target notes. Click on one of them to change the mapping.
On the bottom right there is the list of possible samples. Cick on one of them to change the mapping.

All – Select all maps

None – Deselect all maps

Oct – Select the maps for the current octave

Change Tune/Move – The arrows and dropdown allow to modify the selected notes, or all notes at the same time.
Change tune only alter the target note. Move shifts the mapping down or up for note, sample or both.

Edit mapping/show usage – In edit mode, when clicking on the lists on the right side, the mapping will get altered. If in show mode, instead of editing the mapping, it will show which maps use it.

Set default – Sets the default settings of note and sample for the note mapping of this instrument.

Amplitude Tab

The amplitude tab allows to change settings related to the volume. The envelope works just in the same way than the pitch envelope but it applies to the volume.

Global Volume – Indicates the global attenuation that will have all the notes played with this instrument.

Swing (Randomize) – Percentage of randomization of volume on new notes. The higher the swing, the more it can deviate from Global Volume.

Fadeout – Duration of the fadeout. Fadeout is triggered when a noteoff is called and the envelope doesn't stop, or when the note fadeout NNA is used.

Pan Tab

The pan tab allows to change settings related to the stereo pan. The envelope works just in the same way than the pitch envelope but it applies to the pan.

Instrument pan check – Indicates if new notes of this instrument will use the specific pan, or will obey the channel pan.

Instrument pan slider – If enabled, indicates the pan that new notes of this instrument will use.

Swing (randomize) – Percentage of randomization of pan on new notes. . The higher the swing, the more it can deviate from Instrument pan.

NoteMod center note – If note mod amount is not zero, this is the note that will have the centered pan.

NoteMod amount – If it is not zero, new notes will have a pan value depending on the note that is playing, the note mod center note, and this percentage.

Filter Tab

The filter tab allows to change settings related to the filter. The envelope works just in the same way than the pitch envelope but it applies to the filter.

Filter type – This allows you to set the filter type. You have a choice of Low Pass (retains only the frequencies below the cufoff), High Pass (retains only the freqencies above the cutoff), Band Pass (retains only a band of frequencies above and below the cutoff) and Notch (rejects only a band of frequencies above and below the cutoff). The "Lowpass/IT" filter is also a lowpass, which emulates better the one in "Impulse Tracker".
To ignore any filter settings, set this to "None".

Filter cutoff – Sets the cutoff point for the filter.

Filter – Sets the ressonance or bandwidth of the filter.
