Instrument Editor

Use the arrow buttons to switch between samples, the textbox to rename them, and the "Kill" button to delete them.

Instrument Properties

New Note Action – Whether to cut, release or do nothing when another note from the same machine is triggered while the previous note is still playing.

Panning – The left or right channel bias of the sample. 128 is the middle.

Random Panning – Randomises the panning each time the sample is played.

Random VCF Cutoff – Randomises the filter cutoff each time the sample is played.

Random Reso/Bw. – Randomises the resonance of the filter each time the sample is played.

Tempo Looping Tool

Play sample to fit X rows – Stretches or squashes the sample to fit a given number of rows.

Instrument Wave Layer

Volume – Volume of the sample.

FineTune – Finetuning of the pitch of the sample by a maximum of +/- one semitone.

Tune – Sets the base tuning of the note. The original pitch of the sample is set to C-4, so if the tuning of a given sample were set to C-5, the pitch of the sample is shifted up by one octave.

Loop – To turn on the loop set in the wave editor, click "Forward". Click the "Off" button to ignore the loop settings.

Loop At – Displays the loop points that have been set in the wave editor.

Amplitude/Filter Envelopes

(click on the Amplitude/Filter Envelopes button)

Amplitude – This modifies the ADSR (Attack-Decay-Sustain-Release) envelope for the volume, which is shown on a graph. To ignore the envelope settings, set the Attack, Decay and Release to the lowest value possible, and set the Sustain to 100%.

Filter – This modifies the ADSR envelope for the filter.

Filter Parameters – This allows you to set the filter cut-off frequency, the Q (resonance) and how much the filter envelope affects the sample.

Filter Mode – This allows you to set the filter type. You have a choice of Low Pass (retains only the freqencies below the cufoff), High Pass (retains only the freqencies above the cutoff), Band Pass (retains only a band of frequencies above and below the cutoff) and Notch (rejects only a band of frequencies above and below the cutoff). To ignore any filter settings, set this to "None".
