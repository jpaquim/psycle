Tweaking Machines

The twk (tweak) and tws (smooth tweak) commands allow the parameters of machines to be automated. The twk commmand causes the parameter change to happen instantaneously, while tws interpolates the change from the current value to the new value, over the time of one line.

To insert a twk command, make sure the cursor is in the left-hand side of the track, and press " ` " (the key to the left of the "1" key).
To insert a tws command, do the same but hold the Shift key.
(Please. Note that you can change these keys from the keyboard configuration dialog)

twk 0A 00 A9B3
 1     2    3     4

1. Can be twk or tws.
2. Parameter number.
3. Machine number.
4. Parameter value.

The above example tweaks generator 0 (00), altering parameter 10 (0A) to a value of A9B3 (43443 in decimal).

To find the Parameter number and value for machine you can select "Params" in the third list box from the left in the Machine Bar, and then a list of parameters will display in the fourth listbox.

Alternatively, for a native machine, right click the knob in the parameter window. For a VST, use the listbox that appears via the "View-Parameter List" option. For recording tweaks in real-time, see the Recording section.
