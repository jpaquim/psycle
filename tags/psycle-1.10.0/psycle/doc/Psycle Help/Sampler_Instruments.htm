Instruments

An instrument is essentially a sound sample (currently in .wav and .iff file format) that is used in the internal sampler just like any other generator. The current version of the internal sampler allows you to modify various properties of the sampler without using other external programs, and has its own basic wave editor as well.

All of the instrument functions can be accessed from the right half of the machine bar. Once you've added a sampler using the Add Machine dialogue window, to work with instruments you should make sure that the machine is selected, and that "Instrum" is selected in the third drop-down box.

The next drop-down box dispays the instruments that have been loaded (which are then stored within your psycle file). They are numbered (in Hexadecimal) from 00 to FE (i.e. you can have a maximum of 255 different samples in your song).

Instrument Settings

Load – Loads a sample file from disk into the currently selected instrument position.

Save – Saves the currently selected instrument to disk as a '.wav' file.

Edit – Opens the Instrument Editor window.

Wave Ed – Opens the Wave Editor.
