Pattern properties

(Right Click the Pattern Editor > Pattern Properties)

Here you can name your pattern, which will be displayed on the left-hand side of the status bar,or in the sequence, if you use the Show pattern names option.

Patterns lengths are limited to 1024 lines. When changing the pattern length, ticking "Adapt content to new size" will scale the pattern's contents to match the new length. This is handy when changing the Lines Per Beat (originally starting at a value of 4 and later needing a value of 8, or other similar cases).

The list on the right side allows to change the track name. Select one and modify the value using the editbox above. The change is immediate.

The option "show on pattern header" allows switching between showing the skinned pattern track, or the name of the track. As the description says, this can be also switched by clicking the text "Line" on the top left of the pattern editor.

There is also the possibility to use the same track names for all patterns or have pattern names set different for different patterns.
In such case, there is the option to copy the track names from another pattern into the current one.
