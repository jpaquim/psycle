The Master Machine and Minimixer

(Double click the master machine)

Any machine from which you want to produce audio output must be connected to the Master Machine by a wire (or via another machine). You can have up to 12 connections to the Master, the names of which are listed in the right-hand panel. The levels of the various inputs (i.e. the wire volumes) can be adjusted using the sliders like a conventional mixer.

The view contains a stereo RMS vu-meter and a box where the current maximum (over the last second) values or the peak amplitude and RMS amplitude are shown.

Select the "Decrease Master Volume automatically when Clipping" checkbox to keep your levels below zero decibels (this will prevent digital clipping, which can ruin your song and your speakers).
