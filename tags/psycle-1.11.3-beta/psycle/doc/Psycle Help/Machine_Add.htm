Adding Machines

(F9 or double click the background of the Machine Editor or the new machine button of the toolbar or View > Add Machine)

This opens the New Machine dialogue (Machines can also be added from the Gear Rack, the Machine properties and from the Wire dialog ).
Gear Browser: Displays all the plugins that are available. Select one and click OK to add it to the Machine Editor.
Machine properties: Displays the machine's details, and allows you choose whether to display a warning if a dangerous plugin tries to load.
List machines by: Displays the plugin machines in the Gear Browser by the Plugin Name or by the DLL filename.
Group machines by: Groups the machines depending on its role (Generators, Effects) or host type (Internal, Native, VST)
Load new blitz and gamefx: This is a song loading compatibility option.
New versions of gamefx and blitz are incompatible with the old ones (settings work differently and has a sligtly different sound) so now they have a different dll name.
The option tells the loader to use the new dll instead of the old one. It should only be necessary for songs made with the 1.8.8 beta, or with the unofficial release of the plugin.
Once the psy file is saved again, the dll name will be either the older of the newer and the option will not matter.
Regenerate plugin cache: Psycle uses a caching mechanism to store the names and location of the different machines that it can use. With this button, the file is deleted and completely regenerated.
Check for new plugins: Every time Psycle starts, it removes from the cache those plugins that do not exist anymore, or which dll has been modified. It does not automatically scan for new plugins. This option allows scanning for new plugins keeping the existing cache.

Editing Machines

Double click the machine to bring up its Parameters window or press Shift+Enter. Alter the properties of the machine by right clicking it.

Removing Machines

To remove a machine, right-click the machine and click the "Delete Machine" button (Machines can also be deleted from the Gear Rack ).
