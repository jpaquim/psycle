Sampler Properties

(Double click the sampler machine in the Machine View)

Polyphony – Set the maximum number of notes that the sampler can play at once - the smaller the polyphony, the smaller the memory usage.

Sample Interpolation Method – Select the method that the Sampler uses to play sounds at different to the original (by speeding up or slowing down the sample).

C4 Plays the default speed – This option tells which note will play the sample at the original speed. Originally, the machine used C3 instead of the more usual C4.
