Pattern Settings

Song Bar

Tracks – Number of tracks for your patterns. This is a global setting, and if you reduce your track size, any data loss is irreversible.
Tempo – Speed of the track in beats per minute, ranging from 33 to 999 BPM. You can tweak this value with the Global commands.
Lines per beat – Number of lines per beat, ranging from 1 to 31.
Octave – Current octave, ranging from 0 to 8. On the PC keyboard, this will correspond to the lower octave i.e. with an octave setting of 4, the lower octave will be 4 and the upper will be 5 (see the Keyboard Layout).


Machine Bar

Pattern Step – Select how many lines the cursor moves after a note is entered (1-16). To make this apply only for entering notes (not for movement with the arrow keys, use the option in the Configuration window.
Select Machine – Select the machine to be used.


Sequencer Bar

Follow Song – Make the cursor follow the song progress during playback.

Record NoteOffs – Record Noteoffs (when the key is lifted) during real-time recording.

Record Tweaks – Record any mouse tweaks (altering of parameters) during playback. For more detail on real-time recording, see here.

Show Pattern Names – Shows the pattern name in the sequence, instead of its index. You can put the mouse over the line to show the full name, if it is longer than the list width.
Finally, you can also Right click on it to change the name. Select another pattern to accept the changes.

Multichannel Audition – Allows multiple notes to be played at the same time while entering them (or trying out a synth in the Machine View).

Allow Notes to Effects – Allows notes to effects machines; This can be used with certain VSTs.

Move Cursor When Paste – Moves the cursor to the end of the selection after pasting, instead of remaining in its preview position.
