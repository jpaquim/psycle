Keyboard and Misc. Tab

(Configuration > Settings... >>Keyboard and Misc.)

To change the keymap settings, select the action in the listbox, click inside the "Key:" textbox, then press the new key combination.

Behaviour

Right CTRL = play.
Edit Toggle = stop:		 Causes the right Control key to be used to begin playback from the current sequencer position, and the Record button/key to stop playback.
FT2 Style Home/End Behaviour: When unchecked, pressing home/end will move the cursor to start/end of row (twice), start/end of track and start/end of pattern (3 times),When checked, pressing home/end will move the cursor to the beginning/end of the track.
FT2 Style Delete Behaviour: This applies to the "Delete Row" command (usually backspace). FT2 style deletes the note above cursor, while normal style deletes the note that the cursore is on.
Windows Behaviour on selected blocks This function enables the selections to act like they would do in a Word processor in windows. You can select them, drag them to move them, and when clicking they get unselected. Else, they will behave like in classic trackers.
Use Shift+Arrows for Selection: Allows you to use shift and arrows to make selections in the pattern editor (much like in a word processor).
Wrap Around: When checked, this allows the cursor to go back to the top of the pattern after reaching the bottom (when navigating with arrow keys).
Center Cursor: Keeps the cursor in the centre of the screen in the pattern editor, horizontally and vertically.
Cursor Always Down: Cursor always moves downward (instead of sideways as it normally does in some columns).
Record Mouse Tweaks as tws (Smooth Tweaks): Records tweaks made in real-time as smooth tweaks.
FT2 Style Record Behaviour in Unarmed Track: If there are no tracks with recording enabled, and you play notes/enter commands during playback, with this option data will be entered where the cursor is.
Pattern navigation ignores step (Always jumps one): Ignores pattern step setting and moves down one row in the pattern editor when navigating using the arrow keys.
Page up/page down step by Lets you configure the jump that page up and page down keys will do in pattern lines.

Miscellaneous Options

Autosave Current Song After (mins): Automatically saves the song to "autosave.psy" (in your Psycle Files directory) after the specified number of minutes.
"Save File?" Reminders on Load, New or Exit: Asks the user if the current unsaved file is to be saved upon loading or exiting.
Show Song Info on Load: Displays the Song Properties information when the song is loaded.
Default New Pattern Lines: Sets the default number of lines per pattern (between 1 and 256).
Allow multiple instances This option lets you specify if you would like multiple psycle windows, or would want just one.
Bar highlighting (beats/bar) Use this option to setup the highlighting of a bar in the patterns. ( Beats change when changing lines per beat).
