Sampulse Properties

(Double click the sampulse machine in the Machine View)

Sampulse is one of the two Psycle internal machines that are able to play sampled sounds.
Sampulse is the more powerful of both, and tries to be as feature packed and compatible with the engines in classic trackers like Fasttracker and Impulse Tracker.

Channel settings

This window can be used for two different things, depending on the "Show values" setting.
"Channel" means to setup the default values for the controls in each channel.
"Playback" shows the changing values that the controls have during playback. Controls can also be changed in this mode, but will only affect to the playing voice, not to the channel setup.

Filter Cutoff – Filter default cutoff value for the channel. It will apply if the instrument doesn't have filter by itself, or if there isn't any filter command that changes this default. In playback mode, this will show the final cutoff (including instrument filter, as well as the envelope)

Filter Ressonance – Filter default ressonance value for the channel. It will apply if the instrument doesn't have filter by itself, or if there isn't any filter command that changes this default. In playback mode, this will show the final ressonance (including instrument filter, as well as the envelope)

Panning – Default pan position value for the channel. It will apply if the instrument or sample doesn't have pan or surround by itself, or if there isn't any pan or surround command that changes this default. In playback mode, this will show the final pan (including instrument and sample pan, as well as the envelope)

Surround – Indicates if surround for this channel is enabled by default. It will apply if the instrument or sample doesn't have pan or surround by itself, or if there isn't any pan or surround command that changes this default.

Mute – Indicates if this channel is mute. Muted channels still play, but their audio is not mixed.

Volume – Indicates the default volume of this channel. Volume is affected by multiple settings, one of them being the channel volume. Channel volume can be changed with the channel volume command. In playback mode, this will show the final volume (including instrument and sample volume, as well as the envelope)

Channel num/instrument – Number of the channel or, in playback mode, the instrument that is playing

Slider – Use the slider to show more channels.

Playback settings

Resampling Method – Select the method that the Sampler uses to play sounds at different to the original (by speeding up or slowing down the sample).

Max Voices Playing – Set the maximum number of notes that the sampler can play at once.

Use Amiga slides – This option is meant for the portamento/pitch slide commands, to indicate which method of sliding is desired. The Amiga slide mode, from the original trackers slide by period, which means that slides faster at higher notes than at lower notes.
Modern trackers had an option to slide linearly (i.e. the same amount, no matter which is the current note).

Enable Filters – This option enables or disables the usage of frequency filters. If disabled, filter commands and settings will not have an effect.

Panning mode – When doing stereo pan, there exist several ways to move from full left to full right

  • Linear (Cross), is like drawing a cross (X), so while left volume decreases, right volume increases. (Like IT and ST3). This way, center volume means that left and right play at half the volume
  • Two Sliders (FT2) is like having two independent sliders, similar to FT2 (or similar to the fader control of a DJ mixer). When panning from left to right, first the right volume is increased until it reaches full volume, and then left decreases. This way, center volume plays both left and right at full volume
  • Equal power is similar to Linear, but instead of decreasing linearly, it is decreased logaritmically. This means that center volume is not half (-6dB) but a bit higher (-3dB)
