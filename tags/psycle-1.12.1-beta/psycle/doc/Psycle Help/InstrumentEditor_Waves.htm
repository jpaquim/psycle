Samples bank for Instruments

Use the list on the top left to navigate between the different samples.


The list on the bottom left allows to see which instruments use this sample, if any

Samples bank List Buttons

Load – Loads a sample file from disk into the currently selected sample position.

Save – Save the currently selected sample to disk.

Duplicate – Creates a new sample with a copy of the currently selected one

Delete – Deletes the current sample.

Wave Properties

Wave name – Edit the sample name

Sample rate – The sampling rate of this sample.

Default volume – The volume command used with new notes, if no volume is specified in the pattern.

Global volume – Gain applied to all notes of this sample. It can amplify if needed, so there's no need to amplify the wave data.

Pan position check – If enabled, all the notes that use this sample will play it at the specified pan position. Else, instrument pan or channel pan will be used.

Pan position slider – Pan value to use for this sample if the option is enabled.

Sampled note – If this is the sound of an instrument, which note was sampled, so that it will play the correct note in the pattern.

Finetune – Fine tuning in cents needed to set the correct pitch.

Vibrato Properties

Autovibrato options. They only work with waves that are used by Sampulse, not with the classic sampler.

Waveform – Which waveform will have the oscillator that simulates the vibrato.

Attack – Delay since note start until the vibrato effect reaches full amplitude.

Speed – Speed of the vibrato.

Depth – Amount of the vibrato.

Loop Properties

The bottom right shows a visual representation of the sampled wave, as well as the loop points

Continuous loop – Disabled, forward or pingpong. Indicates if the continuous loop is enabled, and how will it play. Ping pong means that when reachind the end, it runs backward until start and loops again.

Start point – Start point for the loop.

End point – End point for the loop. If the same than sample length, then, the sound is interpolated between that end point and the loop start.

Sustain loop – Disabled, forward or pingpong. Indicates if the sustain loop is enabled, and how will it play. Ping pong means that when reachind the end, it runs backward until start and loops again. Sustain means that it will stop looping when receiving a note off

Start point – Start point for the loop.

End point – End point for the loop. If the same than sample length, then, the sound is interpolated between that end point and the loop start.
