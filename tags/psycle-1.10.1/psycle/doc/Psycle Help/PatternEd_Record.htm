Recording in real-time

This section of the help file talks about entering data into the patterns in realtime. If you look for recording the audio in real-time, see the Record Wav button in the toolbar.

This operation requires that Pattern recording is selected in the toolbar.

To record multiple notes at once (e.g. chords), you need to "arm" the current track - in other words, select the record button on the header of each track (The orange led in the default skin). Do this for every track that you want the notes/tweaks/mcm commands to appear in.

If you only want to record notes in one track, you don't need to arm the track, provided the "FT2 Style Record Behaviour in Unarmed Track" option is ticked in the Configuration window.

Then, to record, press play (F5) and play the notes with either the PC or MIDI keyboard or move the knobs on your machines.
Remember that the Record Tweaks checkbox on the sequencer bar must be ticked for tweaks to be recorded, and that Record noteoff is needed to record the release of the note.
