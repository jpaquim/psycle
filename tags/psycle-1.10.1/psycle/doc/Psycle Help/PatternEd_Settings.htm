Pattern Settings

Song Bar

Tracks – Number of tracks for your patterns. This is a global setting, and if you reduce your track size, any data loss is irreversible.
Tempo – Speed of the track in beats per minute, ranging from 33 to 999 BPM. You can tweak this value with the Global commands.
Lines per beat – Number of lines per beat, ranging from 1 to 31.
Octave – Current octave, ranging from 0 to 8, when entering notes with the PC keyboard. (see the Keyboard Layout).


Machine Bar

Pattern Step – Select how many lines the cursor moves after a note is entered (1-16). To make this apply only for entering notes (not for movement with the arrow keys, use the option in the Configuration window.
Select Machine – Select the machine to be used.


Sequencer Bar

Follow song – Make the cursor follow the song progress during playback.

Show pattern names – Shows the pattern name in the sequence, instead of its index. You can put the mouse over the line to show the full name, if it is longer than the list width.
Finally, you can also Right click on it and use the rename option to change the name. Select another pattern to accept the changes.

Record note-off – Record Noteoffs (when the key is lifted) during real-time recording.

Record tweak – Record any mouse tweaks (altering of parameters) during real-time recording. For more detail on real-time recording, see here.

Multitrack playback on keypress – Allows multiple notes to be played at the same time while playing with a PC keyboard.

Allow notes to effect machines – Allows sending notes to effects machines while using the PC keyboard. This can be used with certain VSTs. Otherwise, notes will be sent to the selected generator.

Move cursor when pasting – Moves the cursor to the end of the selection after pasting, instead of remaining in its preview position.
