Input/Output

(Configuration > Settings... >>Input/Output)

Audio I/O

Depending on your system and soundcard, you will get several different options for sound output. By clicking the "Configure..." button, you can then set the sample rate and turn dithering (essentially waveform "smoothing") on or off. You can set the number of buffers and their size, which will determine the latency (the delay in playback). If you find your audio skips and breaks unnecessarily, you may need to increase the latency.

Note: When using DirectSound, there is an "exclusive" checkbox that, when ticked, will only allow sound to be played when Psycle is in focus.

MIDI I/O

Two MIDI input options are available to you in Psycle: MIDI command and note input using a MIDI keyboard (or other device), or MIDI sequencing using another sequencer (the latter is covered in the MIDI Tutorial).

To use a MIDI Keyboard to input notes, select your keyboard in "Input Device" and make sure that the "Enabled: Sequenced with MIDI Commands" checkbox is cleared.

NOTE: Currently, MIDI Input is not available with Directsound, due to sincronization issues.
