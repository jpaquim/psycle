Directories

(Configuration > Settings... >>Directories)

Song directory:Default directory for opening/saving of .psy files.
Instrument directory:Default directory for samples (.wav and .iff).
Plug-In directory:Directory that contains your native plugins.
VST Plug-In directory:Directory that contains your VST plugins.
Skin directory:Directory containing your Psycle skins.

A word of advice: keep your native plugin directory separate from your VST directory - you can run into problems since both types of plugin are .dll files.
