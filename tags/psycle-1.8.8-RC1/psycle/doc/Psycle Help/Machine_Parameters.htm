Machine Parameters

(Double click the machine)

Native Machines

The parameter dialog for native machines consists of a window with knobs, with the name and value at its right. The Machine number and the name are displayed in the title bar.

They can be edited by dragging the knob up or down to increase or decrease the value respectively. To make value selection easier, holding the Ctrl key while dragging a knob will make a change 10x smaller, and holding the Shift key will change the knob by the smallest value.
Optionally, you can right click on the knob or on the text. This will prompt an edit box where you can insert the exact value in numbers.
Finally, if you doubleclick a knob. It is reset to its default value.

The menu functions are explained below:

About:This menu is machine dependant. Most of them show an about box and information about the pattern commands that they support..
Reset Parameters:Returns all the parameters to their original state.
Random Parameters:Randomises the parameter values (be careful with your speakers, this can give some crazy results - or no sound at all).
Open Presets Dialog:This opens the Presets dialog, which you can use to store the current setting for later use.

To use the preview checkbox in the dialog, just let a pattern play with that generator and select the preset you want to hear. If you cancel, it will be restored to the values you had previous to open the dialog.

VST(i) Machines

The parameter window of VST's is most of the time machine dependant, so if you have any doubt about its usage, check its manual. Some of them, though, will look like Native plugins. For those, tweaking works like explained above.

The menu functions are explained below:

Programs- Open(Save) Preset:Use this option to load(save) an .fxb or .fxp file containing preconfigured programs for the machine. .fxp will be loaded(saved) over(from) the currently selected Program.
Program-Select Program:Use this menu to select the active program. You can alternatively use the < or > buttons in the menu, the Combobox, or the arrows next to the combobox. Select Bank only shows the Internal bank in this version.
Program-Randomize:Randomises the parameter values (be careful with your speakers, this can give some crazy results - or no sound at all).
Programs-Activated:You can use this option to stop or bypass this generator/effect. Alternatively, you can use the Power button in the toolbar.
View-Parameter List:This option shows a window with a list of all the parameters of the VST plugin of which their value can be set. The slider next to the list allows to change the value of the selected one. And you can use it also to know the parameter Index to use for tweaking from the Pattern Editor.
View-Bank Manager:This lets you manage the programs of this machine ( NOTE: Currently it uses the old Preset window to allow using previously saved presets. The dialog is going to change)
View-Midi Channels:Currently disabled. This will allow to setup at which channels this instrument listens to, remap channels, and remap control codes.
View-Toolbar:This shows or hides the toolbar.
View-About:Shows information about this plugin.
