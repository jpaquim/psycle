Using the Sampler in Patterns

For the internal sampler, the note format is as follows:

C-4 01 00 0C23
 1     2    3     4

1. Note name (for more detail, see Entering Notes).
2. Instrument number
3. Machine number (in hexadecimal)
4. Sampler Parameters (see below).

Sampler Parameters

01xx    Portamento Up
02xx    Portamento Down
08xx    Set panning position
09xx    Set sampler offset
0Cxx    Set volume
0ECx    Note cut in x frames
0EDx    Note delay in x frames
15xy    Retrig Command.
y = Number of retrigs, starting at 1
x changes the volume in the following way:

0:No change8:No change
1:+19:-1
2:+2A: -2
3:+4B: -4
4:+8C:-8
5:+16D:-16
6:x 2/3E: x 3/2
7:x 1/2F: x 2
