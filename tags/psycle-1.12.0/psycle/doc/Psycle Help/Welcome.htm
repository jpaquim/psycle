Welcome

Welcome to Psycle!

Psycle is a free, Open-Source, music creation program that offers an easy way to create your own music, fast, and with high quality.

Psycle uses a classical tracker interface (a text grid of notes which are sequenced one after the other), coupled with modularity (plugins) that allow to extend the sounds that you get from it.

Notes are entered in patterns, where you can also use commands (plugin specific, or global), as well as automating parameters via the mcm and twk commands. These patterns are then organized in a linear sequence, which becomes the song order.

It supports four ways to get new sounds: Its own native plugins, the VST Plugin standard, scripted plugins using lua language, and recorded .wav samples using one of the two sampler machines: the sampler machine and the Sampulse machine.
Also, since version 1.8.5, Psycle has a wave-in machine so that it can process live sound.

If this is your first time using Psycle, we recommend reading the Tutorial. For more advanced help, browse through the sections for each part of the interface.

Psycle is developed by the Psycledelics community, based at http://psycle.pastnotecut.org. If you're interested in being involved in Psycle development, read the Become Involved section.
