Using virtual Generators for instruments

Psycle is made around the idea of machines, either generators or effects. They are the elements present in the machine view, and their index number is needed when adding notes in the patterns
When using sampled instruments, they always need a machine to be played, and this makes using sampled instruments a bit different than using other generators, because the pattern needs more information.
To simplify this, and be able to use instruments as a generator in the patterns, Virtual Generators were added.

Virtual generators can be configured in the instrument editor

or the Sampulse version

Virtual Generators have an index number from 81 to FE, and this index is associated to a pair of "Sampled instrument" and "Machine index". In essence, a Virtual generator is an alias for a sampler number and instrument number.

But a Virtual generator is not only an easier way to use a sampled instrument. It also offers another great feature: a volume column.
Since the auxiliary column remains unused with a virtual generator (because the instrument index is already known), it is possible now to add a command here that will get translated as the 0Cxx command if the machine is a sampler, or as the 1Exx command, if the machine is Sampulse.

So, if 80 is mapped to Sampulse 00 and instrument 01 then, the command on the left is the same than the one on the right

C-4 25 80 ....C-4 01 00 1E25

Exchanging virtual generators

One good point of using virtual instruments is that this way it is easy to change which sampler machine it uses for output, or even have two virtual generators to use the same instrument with different samplers (which could have different routes for their wires)

On the other hand, it is not as easy to replace a sampled instrument by another generator, because virtual generator numbers are not selectable from the Gear rack

In order to help on this, the search and replace option has been improved and searches can replace multiple fields at a time.
It should also be possible to change from virtual instrument to sampler as well.
