Psycle Files

New ( File > New ) – creates a new Psycle file.

Open (File > Open) – opens a Psycle File or imports tracker module files (e.g. .xm, .it). All instruments (samples) are converted into Psycle instruments, and sequenced with Sampulse.

Save (File > Save) – saves the current Psycle file.

Save As (File > Save As...) – saves the Psycle file that is currently open under a new name.

Import... Patterns (File > Import > Patterns) – Allows to import the content of patterns so that they can be reused or altered

Import... Instruments (File > Import > Instruments) – Allows to import instruments directly from other songs.

Import... Machines (File > Import > Machines) – Allows to import machines directly from other song.

Export... Patterns (File > Export > Machines) – Allows to export the content of patterns so that they can be reused or altered

Export... Instruments (File > Export > Machines) – Allows to export the instruments to more common formats (.xi, .iti). Wav is already available in the samples save button

Export... Modules (File > Export > Machines) – Exports the song into a Fasttracker 2 "XM" file. Only the patterns and the samples are exported. Placeholders for generator machines are also added, but they will produce no sound by themselves. In the future, Impulse tracker IT might also be added to support more format features.

Render as Wav... – Shows the render to wav dialog to save the song or part of it as uncompressed audio

Song Properties – Allows you to enter a name for the song, the author and commments.

Since version 1.11, it is also possible to change from here the tempo and lines per beat, as well as setting ticks per beat and Extra ticks per line.
These two are used for Sampulse commands, to emulate properly the ticks that commands such as volume slide use.
The ticks per beat is the amount of ticks for each beat. The Extra tick is meant to alter this so that speeds like "5" and "7" can be properly emulated.
Since extra ticks alter the line duration, "Real tempo" and "Real ticks per beat" is also shown.

Revert to Saved – Reloads the file, rejecting any changes made since it was last save. This is IRREVERSIBLE.

Recent files (File > Recent files) – Loads a song from the list of the last opened songs.
